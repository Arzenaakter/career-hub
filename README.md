## ✨ Overview

**CareerHub** is a frontend-based job portal where users can explore job opportunities, view detailed job descriptions, and apply for positions seamlessly.

The platform focuses on **clean UI, smooth navigation, and interactive data visualization**.

---

## 🚀 Key Features

### 🌐 User-Friendly Interface
- Responsive modern design 📱
- Clean and structured layout
- Smooth navigation experience

---

### 🧭 Navigation System
- Dynamic Navbar with:
  - 🏠 Home
  - 📊 Statistics
  - 💼 Applied Jobs
  - 📝 Blog

---

### 🏠 Home Page Experience
- 🎯 Hero Banner section
- 📂 Job Category List
- ⭐ Featured Jobs section
- Real job data display

---

### 💼 Job Listings & Details
- Featured jobs with **“View Details”** button
- Dedicated **Job Details Page**
- Displays:
  - Job description
  - Requirements
  - Additional information

---

### 📌 Job Application System
- 🟢 Apply Now button
- Stores applied job data in **localStorage**
- Instant feedback without backend

---

### 📋 Applied Jobs Tracking
- View all applied jobs in one place
- Organized job list for easy tracking

---

### 📊 Statistics Dashboard
- Interactive charts using **Recharts**
- Displays assignment marks
- Tooltip-enabled data visualization 📈

---

### 📝 Blog Section
- Collection of Q&A content
- Helpful insights and explanations
